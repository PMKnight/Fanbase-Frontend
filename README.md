# Fanbase-Frontend
FanBase is an app designed to give users the ability to share the fan experience with the users community. Users can upload photos and describe what makes this photo a special fan memory.

### User stories
# As a user, I would like to be able to post my fan pictures so that others may see them.
# As a user, I would like to post a memory associated with the uploaded picture.
# As a user, I would like to edit my memories so that I may delete pictures posted previously.
# As a user, I would like to edit my memories so that I may delete or add to memory posts.
